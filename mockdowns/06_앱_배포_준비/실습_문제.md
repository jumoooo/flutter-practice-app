# 🎯 6단계 실습 문제: 앱 배포 준비

## 문제 1: 앱 아이콘 설정하기 🎨

### 목표
앱 아이콘을 커스터마이징하기

### 작업 내용
1. **아이콘 이미지 준비**
   - 1024x1024 PNG 이미지

2. **flutter_launcher_icons 설정**
   - pubspec.yaml에 설정 추가
   - 아이콘 생성

### 확인 사항
- [ ] 아이콘 이미지 준비 완료
- [ ] 아이콘 생성 완료
- [ ] 앱 아이콘 변경 확인

## 문제 2: 앱 이름 변경하기 📱

### 목표
앱 이름을 원하는 이름으로 변경하기

### 작업 내용
1. **Android 이름 변경**
   - AndroidManifest.xml 수정

2. **iOS 이름 변경**
   - Info.plist 수정

### 확인 사항
- [ ] Android 이름 변경 완료
- [ ] iOS 이름 변경 완료

## 문제 3: 릴리스 빌드하기 🏗️

### 목표
릴리스 버전으로 앱 빌드하기

### 작업 내용
1. **Android APK 빌드**
   - flutter build apk --release

2. **빌드 파일 확인**
   - build/app/outputs/flutter-apk/app-release.apk

### 확인 사항
- [ ] APK 빌드 성공
- [ ] 빌드 파일 생성 확인
