## 📚 학습 자료 구조

각 단계별 폴더에는 다음 3가지 파일이 있습니다:

### 📖 학습_가이드.md
- **목적:** 이론과 개념 학습
- **내용:** 
  - 핵심 개념 설명
  - 코드 예제
  - 실전 예제
- **사용법:** 먼저 읽고 이해하기

### 🎯 실습_문제.md
- **목적:** 직접 코딩하며 학습
- **내용:**
  - 단계별 실습 문제
  - 힌트와 가이드
  - 확인 사항
- **사용법:** 직접 코드 작성하며 문제 해결

### 💡 해결_가이드.md
- **목적:** 문제 해결 도움
- **내용:**
  - 자주 발생하는 문제
  - 해결 방법
  - 디버깅 팁
- **사용법:** 막혔을 때만 참고

## 💡 학습 팁

### ✅ 효과적인 학습을 위한 조언

1. **순서대로 진행하세요**
   - 각 단계는 이전 단계를 기반으로 합니다
   - 건너뛰지 말고 순서대로 학습하세요

2. **코드를 직접 타이핑하세요**
   - 복사-붙여넣기보다 직접 작성하면 더 잘 기억됩니다
   - 오타를 고치는 과정도 학습입니다

3. **에러를 두려워하지 마세요**
   - 에러 메시지를 읽고 해결하는 과정이 실력 향상에 도움이 됩니다
   - 해결_가이드.md를 참고하세요

4. **주석을 많이 달아보세요**
   - 나중에 다시 볼 때 이해하기 쉽습니다
   - 한글로 주석을 달아도 됩니다

5. **작은 단위로 나눠서 학습하세요**
   - 한 번에 너무 많이 하지 말고, 단계별로 완성해가세요
   - 각 문제를 완료할 때마다 체크하세요
