## 🎯 학습 방법

각 폴더에는 다음 파일들이 있습니다:

1. **학습_가이드.md** - 이론과 개념 설명
2. **실습_문제.md** - 직접 해볼 실습 문제
3. **해결_가이드.md** - 문제 해결 방법 (필요시 참고)

**권장 학습 순서:**
1. 학습_가이드.md 읽기

3. 막히면 해결_가이드.md 참고
