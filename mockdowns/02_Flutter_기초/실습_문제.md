## 문제 1: 기본 위젯 사용하기 ✅

### 목표
Text, Container, Row, Column 위젯을 사용해보기

### 작업 내용
`lib/main.dart`를 수정하여 다음을 구현하세요:

1. **Text 위젯 스타일링**
   - 크기 24, 굵게, 파란색 텍스트 만들기

2. **Container 위젯**
   - 너비 200, 높이 100
   - 파란색 배경
   - 둥근 모서리 (반지름 10)
   - 중앙에 "안녕하세요" 텍스트

3. **Row 위젯**
   - 3개의 아이콘을 가로로 배치
   - 중앙 정렬

4. **Column 위젯**
   - 3개의 텍스트를 세로로 배치
   - 왼쪽 정렬

### 힌트
```dart
// Text 스타일링
Text(
  '안녕하세요',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.blue,
  ),
)

// Container
Container(
  width: 200,
  height: 100,
  decoration: BoxDecoration(
    color: Colors.blue,
    borderRadius: BorderRadius.circular(10),
  ),
  child: Center(child: Text('안녕하세요')),
)
```

### 확인 사항
- [ ] Text 위젯 스타일 적용 완료
- [ ] Container 위젯 생성 완료
- [ ] Row 위젯으로 아이콘 배치 완료
- [ ] Column 위젯으로 텍스트 배치 완료

### 작업 내용

1. **감소 버튼 추가**
   - FloatingActionButton 옆에 감소 버튼 추가
   - 카운터를 1씩 감소시키는 기능

   - 10 미만이면 파란색

```dart

  children: [
    FloatingActionButton(
      onPressed: _decrementCounter,
      child: Icon(Icons.remove),
    ),
    SizedBox(width: 16),
    FloatingActionButton(
      onPressed: _resetCounter,
      child: Icon(Icons.refresh),
    ),
  ],
)

// 조건부 색상

  '$_counter',
  style: TextStyle(
    color: _counter >= 10 ? Colors.red : Colors.blue,
  ),
)
```

## 문제 4: 할 일 리스트 만들기 📝

### 목표
ListView를 사용해 할 일 리스트 만들기

### 작업 내용
다음 기능을 가진 할 일 리스트를 만드세요:

1. **할 일 목록 표시**
   - ListView.builder 사용
   - 각 항목은 ListTile로 표시

2. **할 일 추가 기능**
   - TextField로 입력받기
   - 버튼으로 추가하기

3. **할 일 삭제 기능**
   - 각 항목에 삭제 버튼 추가

### 힌트
```dart
class _TodoListState extends State<TodoList> {
  List<String> todos = [];  // 할 일 목록
  TextEditingController controller = TextEditingController();
  
  void addTodo() {
    setState(() {
      todos.add(controller.text);
      controller.clear();
    });
  }
  
  void removeTodo(int index) {
    setState(() {
      todos.removeAt(index);
    });
  }
  
  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        // 입력 필드와 추가 버튼
        Row(
          children: [
            Expanded(
              child: TextField(controller: controller),
            ),
            ElevatedButton(
              onPressed: addTodo,
              child: Text('추가'),
            ),
          ],
        ),
        // 할 일 리스트
        Expanded(
          child: ListView.builder(
            itemCount: todos.length,
            itemBuilder: (context, index) {
              return ListTile(
                title: Text(todos[index]),
                trailing: IconButton(
                  icon: Icon(Icons.delete),
                  onPressed: () => removeTodo(index),
                ),
              );
            },
          ),
        ),
      ],
    );
  }
}
```

### 확인 사항
- [ ] 할 일 목록 표시 완료
- [ ] 할 일 추가 기능 작동
- [ ] 할 일 삭제 기능 작동

## 문제 5: 반응형 레이아웃 만들기 📱

### 목표
화면 크기에 따라 다른 레이아웃 보여주기

### 작업 내용
다음과 같이 구현하세요:

1. **가로/세로 모드 감지**
   - MediaQuery 사용
   - Orientation 확인

2. **조건부 레이아웃**
   - 세로 모드: Column 사용
   - 가로 모드: Row 사용

### 힌트
```dart
@override
Widget build(BuildContext context) {
  final orientation = MediaQuery.of(context).orientation;
  
  if (orientation == Orientation.portrait) {
    // 세로 모드: Column
    return Column(
      children: [
        Text('위'),
        Text('아래'),
      ],
    );
  } else {
    // 가로 모드: Row
    return Row(
      children: [
        Text('왼쪽'),
        Text('오른쪽'),
      ],
    );
  }
}
```

### 확인 사항
- [ ] 화면 회전 시 레이아웃 변경됨
- [ ] 세로/가로 모드 모두 정상 작동

## 문제 6: 테마 커스터마이징 🎨

### 목표
앱의 테마를 커스터마이징하기

### 작업 내용
`MaterialApp`의 `theme` 속성을 수정하여:

1. **색상 변경**
   - Primary 색상을 원하는 색으로 변경
   - Secondary 색상도 설정

2. **폰트 변경**
   - 제목 폰트 크기 변경
   - 본문 폰트 크기 변경

3. **버튼 스타일 변경**
   - ElevatedButton 스타일 커스터마이징

### 힌트
```dart
MaterialApp(
  theme: ThemeData(
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.purple,  // 원하는 색상
    ),
    useMaterial3: true,
    textTheme: TextTheme(
      headlineLarge: TextStyle(
        fontSize: 32,
        fontWeight: FontWeight.bold,
      ),
      bodyLarge: TextStyle(fontSize: 18),
    ),
    elevatedButtonTheme: ElevatedButtonThemeData(
      style: ElevatedButton.styleFrom(
        padding: EdgeInsets.symmetric(horizontal: 24, vertical: 12),
      ),
    ),
  ),
)
```

### 확인 사항
- [ ] 색상 테마 변경 완료
- [ ] 폰트 스타일 변경 완료
